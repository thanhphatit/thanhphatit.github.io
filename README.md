## ✨ Overview

**Tools Kit** is a collection of lightweight, fast, and practical web tools built to solve real-world problems.

Instead of relying on multiple websites, this toolkit provides everything in one place — simple, efficient, and easy to use.

---

## 🚀 Key Features

- ⚡ Fast & lightweight (pure HTML, CSS, JavaScript)
- 🧠 Built from real-world needs
- 🎯 Focused on productivity
- 🌐 Runs entirely in the browser
- 🧩 Modular and easy to expand

---

## 📦 Tool Categories

### 🧾 Code & Formatting
- HTML / CSS / JS Beautifier & Minifier
- JSON / XML Formatter & Validator
- JavaScript Obfuscator

### 🔄 Converter & Encoder
- Base64 Converter
- Binary Converter
- URL Encoder / Decoder
- Image to Base64

### 🧠 Generators
- Random Password Generator
- Random JSON / XML
- Random Sentences

### 📝 Text Utilities
- Character Counter
- Remove Line Breaks
- Case Converter
- Vietnamese Text Tools

### 🎨 UI & Design
- Color Picker
- Box Shadow Generator

### 🔗 Other Utilities
- Direct Link Generator
- Upload Images
- RawGit / Staticaly Helper
- And more...
